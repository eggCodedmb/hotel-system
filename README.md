# Vue 3 + Vite

项目结构

```
├── public
├── src
│   ├── assets          静态资源
│   ├── components      通用组件
│   ├── router          路由
│   ├── store           状态管理
│   ├── utils           工具函数
│   ├── views           页面组件
│   ├── App.vue         根组件
│   ├── main.js         入口文件
├── .gitignore          git忽略文件
├── index.html          入口html
├── package.json
├── README.md
├── tsconfig.json
├── vite.config.ts
```

## Project setup

```
yarn  or yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```
